# <a name="main"></a>**4.3 El panell de navegació**


En el **panell de navegació** (que en versions anteriors s'anomenava **panell d'exploració d'objectes**), situat normalment a l'esquerra, hauíem de limitar l'accés únicament als objectes que ens interessa. Per exemple, pot ser molt convenient tenir formularis per a totes les introduccions i modificacions de dades de totes les taules, i després que els usuaris "normals" no tinguen accés a les mateixes taules. Així podríem evitar manipulacions de les taules, i coses fins i tot pitjors.

La primera aproximació seria ocultar el panell d'exploració. Es fa des de **ARCHIVO → Opciones → Base de datos actual**, com es pot observar en la següent figura.

![](43_el_panell_de_navegaci.002.png)



Una altra possibilitat, seria mostrar únicament el que ens interessa. Ho farem en les opcions de navegació del panell de navegació.
